Задание 1
Условие:
Реализовать замыкание make_reader(data, n), которое при каждом вызове возвращает следующие n байт из источника данных. Источником может быть байтовая строка. При исчерпании данных возвращается None.

Описание проделанной работы:
В ходе выполнения задания я создал функцию make_reader, которая принимает байтовую строку data и размер блока n. Внутри функции определена переменная pos для хранения текущей позиции чтения. Возвращаемое замыкание reader использует nonlocal pos для изменения этой переменной. При каждом вызове оно извлекает срез data[pos:pos+n], сдвигает позицию и возвращает очередной блок. Если позиция достигает длины данных, возвращается None. Такой подход позволяет читать данные порциями без хранения дополнительного состояния вне замыкания.

Задание 2
Условие:
Создать декоратор track_memory для отслеживания использования памяти функциями. Декоратор должен замерять объём памяти до и после вызова функции и выводить разницу в байтах.

Описание проделанной работы:
Для измерения памяти использован встроенный модуль tracemalloc. Декоратор запускает трассировку, фиксирует текущее потребление памяти (get_traced_memory()[0]), выполняет декорируемую функцию, снова замеряет память и останавливает трассировку. Разница выводится на экран. Такой подход позволяет оценить реальный расход памяти на выполнение функции без учёта фоновых операций.
